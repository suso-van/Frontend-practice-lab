# Frontend Practice Lab

A structured frontend learning repository containing hands-on projects built using
**React (Web)**, **Next.js**, **React Native**, and **Flutter**.

This repository focuses on:
- Real-world UI development
- Clean frontend architecture
- Component reusability
- Mobile + Web frontend parity

---

## Tech Stack

### Web
- React.js
- Next.js
- TypeScript / JavaScript
- Tailwind CSS / CSS Modules

### Mobile
- React Native
- Flutter (Dart)
